# Frontend Design (Anthropic) — Examples

Reference when applying the frontend-design-anthropics skill and you need concrete theme or typography ideas.

## Theme snippets

Use these as inspiration; adapt to the product and stack.

**RPG / fantasy**
- Fantasy-inspired color palettes, rich and dramatic
- Ornate borders and decorative frames
- Parchment textures, leather-bound styling, weathered materials
- Dramatic lighting, medieval-inspired serif headers

**Solarpunk**
- Warm, optimistic palette (greens, golds, earth tones)
- Organic shapes mixed with technical elements
- Nature-inspired patterns and textures
- Retro-futuristic typography

**Brutalist**
- Strong contrast, raw type, limited decoration
- Monospace or heavy sans; sparse color
- Grid-heavy or deliberately broken grid

**Luxury / editorial**
- Refined serif + clean sans or high-contrast display
- Generous whitespace, subtle texture or grain
- Muted base with one strong accent

**Technical / dashboard**
- IBM Plex, Source Sans 3, or similar; clear hierarchy
- Dark or high-contrast theme; consistent CSS variables
- Data-dense but scannable; motion only where it aids clarity

## Font pairings (high contrast)

- Display + monospace (e.g. Clash Display + JetBrains Mono)
- Serif + geometric sans (e.g. Playfair Display + a clean sans)
- Variable font across weights (e.g. Fraunces 100–900)
- Editorial headline + readable body (e.g. Newsreader + Source Sans 3)

Pick one pairing per project and use it decisively; avoid mixing many families.
